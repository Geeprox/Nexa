# PRD V1: 高亮笔记与关键词搜索

## 背景
用户希望对回答中的关键信息进行沉淀，避免在长对话中反复滚动查找。

## 目标
- 支持对问题与回答片段进行高亮标注。
- 被标注内容写入笔记模块并保留上下文。
- 支持关键词搜索对话与笔记。

## 非目标
- 语义搜索与向量检索。
- 多人协作与共享。

## 用户故事
- 我选中一段关键回答，保存到笔记，便于后续复用。
- 我通过关键词快速检索历史对话与笔记。

## 功能需求
- 选中任意文本 -> 创建高亮标注。
- 高亮标注写入笔记，附带来源节点与消息引用。
- 笔记支持标题与标签。
- 关键词搜索覆盖会话、节点与笔记。

## 交互流程
- 选中文本 -> 标注 -> 笔记生成。
- 搜索框输入关键字 -> 展示结果列表 -> 点击定位。

## 数据需求
- 标注记录需保存位置信息。
- 搜索索引需覆盖对话与笔记。

## 成功指标
- 标注保存操作在 1 秒内完成。
- 搜索结果返回 < 300ms。

## 风险
- 标注范围与消息内容变更后的对齐问题。

## 未决问题
- 笔记是否支持富文本。
- 标签体系是否在 V1 进入。
